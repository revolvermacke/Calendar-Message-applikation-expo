# Calendar-Message-applikation-expo
An application that auto generates messages based on your personal calendar.

For testing this application you need expo go to run it on your phone.
Backend needs to be running on computer ip with "http" not localhost. (dotnet run --urls "http://0.0.0.0:5000" In terminal.)
Frontend you can run with "npx expo start".

When you are on the same wi-fi on your phone as your computer you are able to run the app and try it out!
